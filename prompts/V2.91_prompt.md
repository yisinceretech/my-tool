🤖 系統提示詞 (System Prompt)

你是一個精通 Web Audio API、Three.js (r128) 以及互動式 UI 開發的資深前端工程師。請幫我開發一個「身體聲音合成器 v2.9.1 (Q版女孩特仕版)」的網頁應用程式。

## 【核心嚴格要求】
1. **單檔完美運行**：請做成「單一個 HTML 檔」，將 HTML、CSS、JavaScript 全部寫在一起。
2. **無干擾體驗**：絕對不要使用 `alert`、`confirm`、`prompt` (放網頁後這些會失效)。
3. **完整不分段**：必須提供完整可以直接複製的程式碼，不要分段或省略，方便直接貼上 GitHub 部署。
4. **零外部依賴**：除了載入 Three.js CDN (必須使用穩定的 r128 版本) 之外，不需載入任何外部 3D 模型檔 (.gltf/.glb) 或音檔，所有視覺與聽覺皆由純程式碼與幾何圖形生成。

## 【版面與 UI 設計】
* **視覺風格**：背景使用充滿活力的「粉紫到粉藍」馬卡龍漸層，讓 3D 角色腳下的陰影自然融入。
* **佈局**：採用黃金比例空間分割。左側為 UI 控制面板（包含半透明深色毛玻璃質感的儀表板），右側留給 3D 畫布。
* **2D 監視器畫布**：在左側 UI 中實作一個原生的 2D Canvas 示波器圖表，繪製由右至左滾動的時間軸波形。
    * **心跳模式**：顯示純白背景+黑色網格的心電圖方格紙，配上深紅色的 PQRST 訊號波形。
    * **腸胃模式**：顯示深色背景與青藍色的連續波塊。
* **控制項目**：
    * 「▶️ 開始監聽 / ⏹ 停止監聽」主按鈕（嚴格的靜默備課模式：停止時必須完全靜音並停止波形渲染，調整參數時不發出聲音）。
    * 營養狀態按鈕：一般狀態、咖啡因刺激、暈糖反應、高纖維消化、精緻糖消化。
    * 手動 BPM 滑桿 (40 ~ 160)。

## 【3D 渲染與防彈級穩定度 (Three.js)】
* **人物設計 (Q版女孩)**：請使用 Three.js 的基礎幾何體憑空生成。她必須有：烏黑長直髮、齊瀏海、水汪汪的大眼睛（包含白色鞏膜與高光反射點）、生動的眉毛、實體溫暖膚色的頭部與四肢。
* **衣服設計**：身穿粉紅色 Puma T恤、淺藍色波浪短褲、白襪與淺藍色運動鞋。
* **核心修正 1 (防彈渲染)**：絕對【不要使用】`transmission` 材質。衣服的透視材質請改用最穩定的 `MeshStandardMaterial` 並設定 `transparent: true, opacity: 0.5`，確保在任何裝置都不會隱形。
* **核心修正 2 (防呆位移)**：絕對【不要使用】攝影機 `ViewOffset`。請直接將 3D 模型實體的 X 軸向右推移（例如 `model.position.x = 1.5`），確保她完美置中於螢幕右半部。
* **核心修正 3 (相容性)**：絕對【不要使用】較新版的 `CapsuleGeometry`。所有需要膠囊體的地方（如眉毛、四肢、鞋子），請全部替換成 `CylinderGeometry` 搭配適當的圓角或拼接，確保在 r128 版本不會報錯。
* **發光器官與互動**：胸口有發紅光的心臟，腹部有發綠光/紅光的腸胃。加入射線檢測 (Raycaster)，讓使用者能直接點擊 3D 模型上的器官來切換聽診模式，未選中的器官會暗下來。

## 【Web Audio API 聲音引擎】
* **心跳聲 (Lub-dub)**：使用低頻正弦波（約 45Hz 與 65Hz）搭配指數衰減 (Exponential Decay) 包絡線模擬第一與第二心音。
* **腸胃聲 (Borborygmi)**：使用白噪音節點通過低通濾波器 (Low-pass Filter, 約 200Hz) 模擬體內共鳴，並以超低頻 LFO (約 3Hz) 進行振幅包絡調變，製造斷續的腸道蠕動感。
* **無縫循環**：必須將音訊緩衝區設為 `loop = true`，實現無限循環播放。

## 【生理反射與連動邏輯 (最重要)】
* **視覺分離 (解耦)**：當聽診器焦點在「腸胃」時，3D 心臟必須停止跳動動畫，化為安靜的背景，讓視覺 100% 專注在腸胃上；反之，聽診心臟時腸胃也要維持穩定狀態。
* **狀態連動 (聲音 + 3D 表情 + 動畫)**：
    * **☕ 咖啡因**：心跳極快 (BPM>100)，心臟/ECG劇烈起伏。3D 女孩會嚇得睜大眼睛成 "O" 型嘴、挑高眉毛、皮膚泛紅、雙手舉起，且全身會有微小的神經發抖 (Jitter) 動畫。
    * **🍩 暈糖反應**：心跳緩慢且帶有不規則延遲。女孩會變成半睜開的無神死魚眼、八字眉下垂、膚色蒼白，頭部疲憊歪斜、雙手無力下垂。
    * **🥦 高纖維**：腸胃發出健康綠光，產生厚實持續的咕嚕聲與密集波形。女孩呈現彎月笑眼，張嘴微笑，頭部開心微晃。
    * **🥤 精緻糖**：腸胃發出警告紅橘光，產生斷續空洞的噪音。女孩眉頭深鎖、眼皮下垂、表情痛苦抽搐。
